# fireplace-cli
Funny Christmas Lights/Fireplace app to bring joy for everyone during this jolly holiday season!

## Building (NixOS):
```
git clone https://github.com/anteroul/fireplace-cli.git
cd fireplace-cli
nix-build
nix-env -i ./result
```

### Running:
Run the application by simply typing the command 
```
fireplace
```
in to your terminal.
